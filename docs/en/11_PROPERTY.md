# 11 — Property

The property mini app provides community-verified sale and rental listings without exposing confidential unit records.

## Capabilities

- Owner/authorized-agent listing with verification
- Unit attributes, approved photos, asking price, availability, occupancy status
- Search, comparison, saved listings, inquiry, and visit scheduling
- Listing lifecycle: draft, review, active, reserved, closed, withdrawn
- Internal historical analytics using privacy-safe aggregated data

## Controls

Ownership documents are restricted; brokers require authorization; public claims are moderated; transaction history is not represented as official valuation unless sourced from an authorized provider.
