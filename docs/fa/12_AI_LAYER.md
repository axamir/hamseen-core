# ۱۲ — لایه هوش مصنوعی

هوش مصنوعی قابلیت کنترل‌شده پلتفرم است، نه چت‌بات تزئینی.

```text
اپ / تلگرام / بله
        ↓
API هسته همسین
        ↓
زمینه مجوز و حریم خصوصی
        ↓
درگاه AI
        ↓
OpenAI / Gemini / مدل محلی / ارائه‌دهنده سفارشی
        ↓
خواندن، پیش‌نویس، پیشنهاد یا اقدام نیازمند تأیید
```

## دستیارهای نقش‌محور

- ساکن: شارژ، رزرو، مهمان، درخواست، خدمات و خلاصه اعلان
- نگهبان: ثبت صوتی مهمان/بسته، مجوز و خلاصه شیفت
- مدیر: صندوق هوشمند، اولویت، پیش‌نویس ارجاع و گزارش روزانه
- حسابدار: تحلیل انحراف، فهرست بدهکار، هشدار تکرار و پیش‌نویس گزارش
- هیئت‌مدیره: خلاصه حاکمیت، تحلیل روند و پیش‌نویس صورت‌جلسه
- خدمات‌دهنده: خلاصه سفارش، کار معوق و بینش تقاضا و موجودی

## مرکز کنترل AI

انتخاب ارائه‌دهنده/مدل، کلید رمزگذاری‌شده، مسیریابی وظیفه، سقف مصرف و هزینه، نسخه پرامپت، دامنه داده مجاز، پایش خطا، کلید قابلیت، تاریخچه حسابرسی و قطع اضطراری.

## ایمنی

- دسترسی AI برابر دسترسی کاربر است.
- پاسخ از داده معتبر همسین بازیابی می‌شود و عدم قطعیت اعلام می‌شود.
- پرداخت، تغییر مجوز، حذف، اعلان رسمی و تصمیم حاکمیتی تأیید روشن انسان می‌خواهد.
- هر درخواست، کاربر، مدل، دامنه داده، خروجی، تأیید و اجرا را ثبت می‌کند.
