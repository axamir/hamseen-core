# مستندات همسین — فارسی

اسناد را به ترتیب شماره بخوانید. مسیر از هدف و زمینه پایلوت آغاز می‌شود و سپس به رفتار محصول، معماری، تحویل و تجاری‌سازی می‌رسد.

| بازه | موضوع |
|---|---|
| ۰۰–۰۳ | مانیفست، محصول، پایلوت و نقش‌ها |
| ۰۴–۱۲ | معماری اطلاعات و ماژول‌های محصول |
| ۱۳–۱۵ | داده، API و امنیت |
| ۱۶–۲۰ | MVP، پذیرش، ساخت، پایلوت و تجاری‌سازی |
| ۲۱–۲۵ | گردش‌کار، UX، طراحی، پیاده‌سازی و لندینگ‌پیج |
| ۲۶ | پرسش‌نامه تأیید داده‌های واقعی برج توسط هیئت‌مدیره |

برای جلسه با هیئت‌مدیره از [پرسش‌نامه تأیید داده‌های لویزان](26_BOARD_DATA_VALIDATION.md) استفاده کنید. مقادیر پیشنهادی این فرم تا زمان تأیید، واقعیت برج محسوب نمی‌شوند.
