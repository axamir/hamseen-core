# ۱۴ — معماری API

## مرز

کلاینت و ربات API نسخه‌دار همسین را صدا می‌زنند و هرگز مستقیم به دیتابیس یا ارائه‌دهنده مدل وصل نمی‌شوند.

## سرویس‌های پیشنهادی

هویت، جامعه، واحد، مالی، ورود، درخواست، اعلان، رزرو، تجارت، املاک، اتصال، درگاه AI و حسابرسی.

## استانداردها

- REST/JSON با `/v1` در ابتدا و قرارداد رخداد برای جریان غیرهمزمان
- کلید idempotency برای پرداخت، سفارش و اقدام ربات
- صفحه‌بندی cursor، کد خطای پایدار و شناسه همبستگی درخواست
- وب‌هوک امضاشده با جلوگیری از بازپخش
- مشخصات OpenAPI و کلاینت تولیدشده
- محدودیت نرخ بر اساس مجتمع، کاربر، کانال و endpoint

## کانال‌ها

PWA/اپ، وب مدیریت، تبلت نگهبان، ربات تلگرام، ربات بله، پیامک/Push، درگاه پرداخت، ذخیره‌ساز شیء و مدل‌های AI.

## رخدادها

نمونه: `guest.waiting`، `guest.approved`، `package.received`، `ticket.created`، `payment.receipt_submitted`، `reservation.confirmed`، `order.status_changed` و `announcement.published`.
